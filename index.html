VPN Setup Guide

Configure secure access to Mensa Brands internal resources

warning
Security Notice

This guide and all associated links are for Mensa Brands employees only. Do not share credentials or access outside the organization. Please maintain the VPN users list in the designated VPN sheet.

1

Create User in Sophos Portal

Log in to the Sophos Portal and create a new user account. Assign the Open SSL VPN policy to the newly created user.

open_in_new Open Sophos Portal
info If the link doesn't open, copy and paste this URL: https://172.16.0.1:4444/webconsole/webpages/index.jsp#10442
2

Set Username & Password

On the same page in Sophos Portal, configure a username and password for the new user.

Important: Share these credentials securely with the user via encrypted email or secure messaging.

3

User Login to Portal

From the user's device, navigate to the User Portal and log in using the credentials provided in Step 2.

login Open User Portal
info Direct URL: https://14.194.88.242
4

Install OpenVPN Client

Download and install the OpenVPN Connect Client on the user's system (Windows/Mac).

download Download OpenVPN Client
5

Download VPN Configuration File

Once logged into the user portal, locate and download the VPN configuration file (.ovpn). This file contains all necessary settings to connect to the Mensa Brands VPN.

description Download Configuration
6

Import Configuration into OpenVPN

Open the OpenVPN Connect application on your system. Import the configuration file by:

  • Double-clicking the .ovpn file, or
  • Dragging and dropping it into the OpenVPN app window
7

Approve Profile Import

A prompt will appear asking you to import the VPN profile. Click OK or Approve to confirm the import.

8

Enter Credentials in OpenVPN

When prompted by the OpenVPN Connect app, enter the username and password created in Step 2. You may choose to save the credentials for future connections.

9

Connect to VPN

Click the Connect button in the OpenVPN app. Once connected, you will have secure access to all Mensa Brands internal applications and resources!

check_circle Success! You're now connected to the Mensa Brands network.

Rediffmail Setup in Outlook

Configure your Rediffmail account in Microsoft Outlook using IMAP

lightbulb
Pro Tip

Make sure you have your Rediffmail email address and password ready before starting the configuration. These are the same credentials you use to log in to Rediffmail webmail.

inbox

Authentication Required ✓

checklist Setup Steps in Outlook

1

Open Microsoft Outlook and go to File → Add Account

2

Enter your Rediffmail email address and click Connect

3

Select IMAP as the account type when prompted

4

Enter the incoming mail server settings as shown above (imap.rediffmailpro.com, Port 993, SSL/TLS)

5

Enter the outgoing mail server settings (smtp.rediffmailpro.com, Port 587, SSL/TLS)

6

Enter your password when prompted and click Connect

7

Wait for Outlook to test the connection and click Done when complete

help_outline Troubleshooting Tips

sync_problem

Connection Failed?

Verify your email and password are correct. Try port 465 instead of 587 for SMTP.

lock

Authentication Error?

Ensure "Require authentication" is enabled for outgoing mail with same credentials.
