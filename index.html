From the user's device, navigate to the User Portal.

Install OpenVPN Client

5

Download VPN Configuration File

Once logged into the user portal, locate and download the VPN configuration file (.ovpn).

This file contains all necessary settings to connect to the Mensa Brands VPN.

üìÑ Download Configuration

Note: Access the user portal first, then download your configuration file from there.

6

Import Configuration into OpenVPN

Open the OpenVPN Connect application on your system.

Import the configuration file by either:

  • Double-clicking the .ovpn file, or
  • Dragging and dropping it into the OpenVPN app window
7

Approve Profile Import

A prompt will appear asking you to import the VPN profile.

Click OK or Approve to confirm the import.

8

Enter Credentials in OpenVPN

When prompted by the OpenVPN Connect app, enter the username and password created in Step 2.

You may choose to save the credentials for future connections.

9

Connect to VPN

Click the Connect button in the OpenVPN app.
